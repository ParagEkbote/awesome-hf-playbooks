#  Awesome HF Playbooks

This repository contains a collection of comprehensive playbooks and guides focused on advanced techniques in machine learning, particularly large language models (LLMs), training methodologies, distillation, evaluation, and synthetic data generation. These resources are curated from Hugging Face to provide practical insights for building and optimizing world-class AI models.

## Resources

### 1. HF Ultrascale Playbook
**File:** `HF_ULTRASCALE_PLAYBOOK.pdf`  
A detailed guide on scaling large language models to ultrascale levels, covering infrastructure, training strategies, and optimization techniques for massive model deployments.

### 2. The Smol Training Playbook: The Secrets to Building World-Class LLMs
**File:** `The Smol Training Playbook The secretes to Building World-class LLMs.pdf`  

This playbook reveals the essential techniques and best practices for training high-quality, efficient small-to-medium language models that rival larger counterparts in performance.

### 3. Distilling 100B Models 40x Faster with TRL
**File:** `distilling-100b-models-40x-faster-with-trl.pdf`  
A comprehensive guide on knowledge distillation techniques using the Transformers Reinforcement Learning (TRL) library, demonstrating how to compress massive 100 billion parameter models into smaller, faster versions while maintaining performance.

### 4. FineVision: Open Data is All You Need
**File:** `finevision-open-data-is-all-you-need.pdf`  
Explores the power of open-source datasets for fine-tuning vision models, providing strategies for leveraging publicly available data to achieve state-of-the-art results in computer vision tasks.

### 5. The LLM Evaluation Guidebook
**File:** `the-llm-evaluation-guidebook.pdf`  
A thorough resource for evaluating large language models, covering various evaluation metrics, benchmarks, and methodologies to assess model performance, safety, and capabilities across different domains.

### 6. The Synthetic Data Playbook: Generating Trillions of the Finest Tokens
**File:** `the-synthetic-data-playbook-generating-trillions-of-the-finest-tokens.pdf`  
This playbook delves into the creation and utilization of synthetic data for training LLMs, offering techniques to generate massive amounts of high-quality training tokens to improve model generalization and performance.

### 7. Unlocking On-Policy Distillation for Any Model Family
**File:** `unlocking-on-policy-distillation-for-any-model-family.pdf`  
Provides insights into on-policy distillation methods that can be applied across different model architectures, enabling efficient knowledge transfer and model compression techniques.
